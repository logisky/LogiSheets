# factory-simulator(中文)

经营你的工厂!一个完全基于 LogiSheets 构建的互动模拟游戏。

它是引擎能力的旗舰演示——用 **Block** 承载结构化状态、**实时公式**驱动模拟、
**craft 交互**(可点击/拖拽的单元格)接收输入。游戏逻辑在
[`factory-simulator-core`](../factory-simulator-core);本包是中文版的 UI 外壳。

### ▶ [Play it live](https://www.logisheets.com/?craft=factory-simulator-zh)

## 构建

```bash
yarn workspace factory-simulator-zh build   # 打包并拷贝到 public/
```
