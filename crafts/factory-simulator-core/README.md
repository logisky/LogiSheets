# factory-simulator-core

Shared engine and locale-parameterized logic for the Factory Simulator crafts.

This is a **source-only** package (no build step): it holds the simulation
rules, balance, and game logic that [`factory-simulator-zh`](../factory-simulator-zh)
and [`factory-simulator-en`](../factory-simulator-en) both consume, so the two
locale editions stay in sync and only differ in their copy.

Not a runnable craft on its own — see the locale editions to play.
