# factory-simulator (English)

Run your factory! An interactive simulation game built entirely on LogiSheets.

It's the flagship demo of what the engine can do — **Blocks** for structured
state, **live formulas** driving the simulation, and **craft interactions**
(clickable, draggable cells) for input. The game logic lives in
[`factory-simulator-core`](../factory-simulator-core); this package is the
English edition's UI shell.

### ▶ [Play it live](https://www.logisheets.com/?craft=factory-simulator)

## Build

```bash
yarn workspace factory-simulator-en build   # bundles + copies to public/
```
